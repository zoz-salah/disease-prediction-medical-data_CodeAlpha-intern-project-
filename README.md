# Disease Prediction from Medical Data

A beginner-friendly machine learning project for predicting diseases (heart disease, diabetes, breast cancer) using patient medical data. Implements multiple classification algorithms with comprehensive evaluation.

## Features

- **Datasets**: Heart Disease (Cleveland), Pima Indians Diabetes, Breast Cancer Wisconsin.
- **Models**: Logistic Regression, Support Vector Machine (SVM), Random Forest, XGBoost.
- **Preprocessing**: Handles missing values, feature scaling, and train-test splitting.
- **Evaluation**: Accuracy, Precision, Recall, F1-score, ROC-AUC, Confusion Matrix, Feature Importance.
- **Prediction**: Single patient interactive mode or batch CSV prediction.
- **Hyperparameter Tuning**: GridSearchCV for each model (optional).
